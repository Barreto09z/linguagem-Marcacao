O que é Tailwind CSS?

Tailwind CSS é um framework CSS "utility-first" que pode ser composto para construir qualquer design, diretamente no seu HTML...
Ao contrário de outros frameworks como Bootstrap, ele não vem com componentes de UI pré-fabricados. Em vez disso, ele fornece classes de baixo nível que você pode usar para construir designs completamente personalizados sem nunca sair do seu HTML.
